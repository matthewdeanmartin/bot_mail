# Security Policy for `bot_mail`

## Introduction

This document outlines how to report vulnerabilities.

## Supported Versions

Security updates will only be applied to the **most recent release** available on PyPI.

| Version | Supported |
|---------|-----------|
| latest  | Yes       |
| older   | No        |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

1. **GitHub private vulnerability reporting**
2. **Email** — contact **`matthewdeanmartin@gmail.com`**
